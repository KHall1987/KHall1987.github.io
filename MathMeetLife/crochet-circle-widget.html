Parametric

Flat crochet circle stitch calculator

A small companion widget for the geometric model behind flat crochet circles. Enter a target circumference or radius, then tune stitch height and width to generate round-by-round stitch counts.

Based on the circle model C = 2πr, with each round adding roughly one stitch height to the radius.

Use one unit system throughout. If h and w are measured in centimetres, enter the target circumference or radius in centimetres as well.

Inputs

Example: 20 cm, 8 inches, or any consistent length unit.
Use any positive values. For example, h = 1 and w = 1 are valid.

Round model: C(N) = 2πNh

Total stitches: n(N) = round(2πNh / w)

From circumference: N = C / 2πh and n = C / w

From radius: N = r / h and n = 2πr / w
