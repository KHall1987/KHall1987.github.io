Choose a sphere, cone, or cylinder, enter size and gauge, then copy a plain round-by-round pattern. The calculator uses the same stitch-count conversion for each shape; only the radius profile changes.

Shared model: stitches ≈ L / w. Joined rounds use L = 2πr; continuous rounds use the local spiral approximation L = sqrt((2πr)^2 + pitch^2).

Model note

Pattern-friendly smoothing is a constrained rounding pass. It keeps the stitch counts near the mathematical target, but only allows ordinary crochet shaping.

Sphere profile: r(s) = R sin(s / R)
Cone profile: r(s) = Rbase * s / S, where S = sqrt(Rbase^2 + H^2)
Cylinder profile: r(s) = R
Joined-round length: L = 2*pi*r(s)
Continuous-round approximation: L = sqrt((2*pi*r(s))^2 + pitch^2)
Raw count: Sraw = round(L / w)
Smoothed counts must satisfy: ceil(Sprev / 2) <= Snext <= 2*Sprev

Most changes update automatically. Use reset if you want to return to the starter settings.
